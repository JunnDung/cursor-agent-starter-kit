# Creating Agents

Agents are reusable role definitions stored in `agents/`.

## When to create an agent

Create an agent when a task pattern repeats often.

Examples:
- code reviewer
- bug fixer
- researcher
- test writer
- refactorer

## How to create an agent

Use:

```txt
.cursor/skills/create-agent/SKILL.md
```

Or copy:

```txt
templates/agent.md
```
