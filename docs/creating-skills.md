# Creating Skills

Skills are reusable task-specific procedures stored in `.cursor/skills/`.

## When to create a skill

Create a skill when a task has a repeatable process.

Examples:
- create an agent
- create a tool
- review code
- debug a bug
- refactor safely
- write tests

## How to create a skill

Copy:

```txt
templates/skill.md
```

Then place it in:

```txt
.cursor/skills/skill-name/SKILL.md
```
