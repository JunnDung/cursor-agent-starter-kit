# Release Workflow

## Goal

Prepare the repository for a clean public release.

## Steps

1. Check README quality.
2. Check examples.
3. Check docs.
4. Check package metadata.
5. Check license.
6. Check broken links.
7. Check tasks and roadmap.
8. Run verification commands.

## Output

```txt
Release readiness:
Blockers:
Recommended fixes:
Verification:
```
